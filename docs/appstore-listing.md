## 概要（4000字以内）

```
「あとで読む」に溜め込んで、結局見ない。
読んだその場ではわかった気がするけど、すぐ忘れる。
知識として蓄積されない——

そんな「もったいない読書」を終わりにするアプリです。


■ KnowledgeBaseAI とは

保存した記事を、Apple Intelligence（完全オンデバイスAI）が自動で分析・要約・タグ付け・概念化します。

あなたはただ「気になった記事を保存する」だけ。AIが裏で働き、複数の記事から「超まとめ（Wikiページ）」を自動生成。知識が静かに、でも確実に育っていきます。

必要なときに、最新の自分の理解として取り出せる——それがKnowledgeBaseAIの存在理由です。


■ 主な機能

• 記事・PDF・メモ・音声・写真を一括取り込み
• AIが自動でタグ付け・カテゴリ分類・要点抽出
• 関連記事を束ねた「超まとめ（Wikiページ）」を自動生成・自動更新
• AIチャットで保存した知識の中から回答
• iCloudでデバイス間同期


■ 3つの特長

【完全無料・完全オンデバイス】
Apple Intelligenceを使用するため、サーバーへのデータ送信ゼロ。あなたの読書履歴・思考はすべて手元のデバイスだけに留まります。プライバシーは機能ではなく、設計の根幹です。

【iPhoneが賢くなるたびに、自動で賢くなる】
Apple Intelligenceのアップデートとともに、このアプリのAIも自動的に向上します。追加料金不要、iOSをアップデートするだけ。

【AIはあくまで「裏方」】
チャットでAIを意識して使う設計ではありません。記事を保存すれば、あとはAIが自動で整理します。あなたはただ「読む」だけでいい。


■ こんな方におすすめ

• 毎日記事やニュースを読むが、記憶に残らないと感じている
• 読書メモがバラバラで活用できていない
• AIツールを使いたいが、プライバシーが心配
• 知識を複利で蓄積したい
• サブスクなしで本格的なAIアプリを使いたい


© 2026 Chia-Wei Chang
```
