# 「あとで読む」をやめました。保存するだけで、AIが“自分専用の百科事典”を育ててくれるアプリを作った話

> iPhone・iPad向けのアプリ「**Knowledge Base**」を紹介します。気になった記事を保存するだけで、AIが端末の中だけで要点を抜き、テーマごとに整理し、あなただけのWiki（百科事典）を育てていく。アカウント不要・サーバーなし・完全オンデバイス。情報を“溜める”のではなく、“育てる”ためのアプリです。

> 📸 **【スクショ①｜ヒーロー画像】** 記事を数十件保存して育った状態の「ナレッジ」タブ（概念まとめ一覧／要点が見えている画面）。**1枚目はアプリの第一印象になるので最重要**。きれいに整った“育った”状態を撮るのがコツ。横幅いっぱいに大きく載せる。

> ℹ️ この記事は `📸【スクショ◯】` の箇所に画像を差し込む前提で書いています。公開時はマーカー行を実際の画像に置き換えてください。撮影リストは記事末尾の「スクショ撮影チェックリスト」にまとめてあります。

## こんな経験、ありませんか？

- 「あとで読む」に放り込んだ記事を、結局二度と開かない
- ブラウザのタブが何十個も開きっぱなし
- 同じテーマの記事を何度も読んでいるのに、知識が頭の中で**つながらない**
- 「あの話、どこで読んだんだっけ？」と探すだけで時間が溶ける

情報を**保存する**アプリはたくさんあります。でも、そのほとんどは“保管庫”で終わってしまう。保存した瞬間に満足して、知識は散らばったまま。

**問題は「保存」ではなく「その後」にあります。**

Knowledge Base は、この「その後」をぜんぶAIに任せるアプリです。

---

## 一言でいうと

> **読んだ記事を、AIが裏で1つの百科事典に編集し続けてくれる。開くだけで“自分だけの知識”が育っていくのが見える、優しい第二の脳。**

あなたがやることは、**気になったものを保存するだけ**。あとはAIが読んで、整理して、まとめてくれます。

## 使い方は、たった3ステップ

### 1. 保存する

SafariやXなど、どのアプリからでも**共有シートで「Knowledge Base」を選ぶだけ**。
記事URLはもちろん、**PDF・写真（文字を読み取り）・音声メモ（文字起こし）・選択したテキスト**まで、なんでも放り込めます。保存は一瞬で終わります。

> 📸 **【スクショ②｜保存】** Safariなどで記事を開き、**共有シートに「Knowledge Base」アイコンが並んでいる**画面。または＋ボタンから「URL / メモ / ファイル / 写真 / 音声」の取り込みモードが見える画面。「どこからでも放り込める」が一目で伝わるものを。

### 2. AIが裏で整理する

保存すると、端末の中のAIが自動で動き出します。

- 記事の**要点・大事な事実・登場する人物やモノ**を抜き出す
- 英語や中国語の記事は、**日本語に翻訳してから**知識化する
- 関連する記事をまとめて、**テーマごとの「概念ページ（Wiki）」** を作る・更新する
- 自動でタグと分野を付けて整理する

あなたは待つ必要はありません。保存はすぐ終わり、整理は裏で進みます。

> 📸 **【スクショ③｜概念ページ／Wiki】** AIが複数記事をまとめた「概念ページ」の詳細（**要点リスト＋本文＋出典記事**が見える画面）。このアプリの“知識が育つ”核なので、関連記事が多くて中身が充実したページを選ぶ。

### 3. 育った知識を、開くだけ／質問するだけ

3つのタブで、育った知識を受け取れます。

- **ナレッジ**：概念ごとの「超まとめ」を、答え先出しで一覧。各まとめの**最重要ポイント**が、開いた瞬間に目に入る
- **ライブラリ**：保存した記事を日付別に。関連度ランキングで検索
- **AIチャット**：「保存した記事の中から」質問に答えてくれる。回答には**番号付きの出典**が付くので、元記事にすぐ戻れる

> 📸 **【スクショ④｜3タブ並べ】** 「ナレッジ」「ライブラリ」「AIチャット」の3画面を**横並び（または3枚連続）** で。特にAIチャットは、質問→**番号付き出典 `[1] [2]`** が付いた回答が見える状態を撮ると、RAGの安心感が伝わる。ライブラリは日付ごとに整理された一覧を。

## ここが他と違う：3つの特徴

### 🔒 完全オンデバイス。あなたのデータは外に出ない

要約も、翻訳も、整理も、AIチャットも、**すべて端末の中だけ**で動きます（Apple Intelligence）。

- 開発者のサーバーに**送信ゼロ**
- アカウント登録・ログイン**不要**
- トラッキングなし・広告なし・解析SDKなし

知識ベースは「自分が何に興味があるか」そのもの。一番プライベートなデータだからこそ、外に出さない設計にこだわりました。同期したい人は、**自分のiCloud（プライベート）** だけで端末間同期できます（任意・既定オフ）。

> 📸 **【スクショ⑤｜プライバシー】** 設定画面のiCloud同期トグル（既定オフ）が見える画面、または **App Storeの「Appのプライバシー＝データを収集していません」** の表示。「ログイン画面が存在しない＝アカウント不要」が伝わる絵でもOK。安心感を一枚で。

### 📚 “記事の山”ではなく“育つ百科事典”

このアプリは、保存した記事を並べるだけではありません。AIが複数の記事を横断して、人物・モノ・概念ごとに**1枚のWikiページ**にまとめ上げます。同じテーマの記事が増えるほど、そのページは賢く・詳しくなっていく。**読むほどに、自分の理解が更新されていく**感覚です。

> 📸 **【スクショ⑥｜分野で整理】** 分野（カテゴリ）で絞り込んだ一覧や、概念カードがいくつも並んだフィードなど、「**散らばった記事が束ねられている**」のが分かる画面。①ヒーローと被らない切り口（カテゴリ別・ドリルダウン）を選ぶと良い。

### 🤖 でも、最後はあなたが主導権を持つ

AIは便利ですが、間違えることもあります（特に音声の固有名詞や分類）。だから、

- 本文の表現を見直す／自然言語で訂正を指示する
- 分類が違ったら、正しい分野に直す（**直すとAIが学習して、次から間違えにくくなる**）
- 概念ページを編集・非表示にする

“AIにお任せ、でも最終判断は人間”。この安心感を大切にしています。

> 📸 **【スクショ⑦｜人が直す】** 「分類の確認」画面（要確認タグに正しい分野を選べる画面）、または本文の「見直し／訂正」シート。「AI任せきりではなく、自分で直せる」が伝わる操作画面を。

ブックマークは増えるほど重荷になります。でも、知識は本来、増えるほど豊かになるはず。

Knowledge Base は、あなたが読んだものを、勝手に、自分だけの百科事典に変えていきます。週に1〜2回開くだけでも、そこにはいつも“最新の自分”がいる。

**保存するだけ。あとは、育つ。**

そんな第二の脳を、ぜひ手のひらの中で育ててみてください。

### 撮り方のコツ
- **端末の枠（ベゼル）付きモック**にすると見栄えが良い（Appleの公式モックや無料のフレーム素材を活用）。
- **ライトモードで統一**するか、ライト/ダークを意図的に使い分ける（混在させない）。
- 個人情報や生々しい保存内容が写らないよう、**見せても良い記事**で揃える。
- noteは横長で大きく表示されるので、**文字が読める解像度**で。スクショは等倍（Retina）のまま貼るのが安全。
- 縦長スクショを並べるときは、**2〜3枚を1枚に合成**すると記事が間延びしません。

### あると強い“番外”カット
- アプリアイコン（記事の見出し画像やSNSシェア用に）
- 音声メモを共有→自動で文字起こし→知識化される一連（取り込みの幅が伝わる）
- 英語記事を保存→日本語の要点になっている（翻訳の価値）
