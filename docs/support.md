# 知積 サポート

知積 (KnowledgeTree) のサポートページです。お困りごと、不具合報告、機能要望はこちらから。

---

## 📞 連絡方法

### 不具合報告 / 機能要望

**GitHub Issues**: [github.com/changch223/KnowledgeTree/issues](https://github.com/changch223/KnowledgeTree/issues)

新規 issue を立ててください。次の情報があると対応がスムーズです:

- 端末モデル (例: iPhone 17 Pro)
- iOS バージョン (設定 → 一般 → 情報)
- 知積アプリのバージョン
- 再現手順
- 期待した動作 vs 実際の動作

### 一般的な質問

GitHub Discussions は近日有効化予定です。それまでは Issues に「question」ラベルでお気軽にどうぞ。

## ❓ よくある質問 (FAQ)

### Q1. 「最近のあなた」セクションが表示されない

**A**: 以下を確認してください:
- 前回知識 Clip タブを開いた時から、新しい記事を保存しましたか?
- 0 件の場合は静かに非表示になります (calm UX、罪悪感を生まないため)
- Apple Intelligence 対応端末ではない場合、Fallback で擬似 3 段落が表示されます

### Q2. 「動的トピック候補」が出てこない

**A**: 動的トピックの自動発見は以下の条件で動作します:
- essence embedding (AI 分析済みデータ) を持つ記事が **10 件以上**
- 前回 batch 実行から **7 日経過** (初回起動時は即実行)
- 最低 **3 件** が同じクラスタに集まる

記事が少ない場合は、もう少し保存を続けてみてください。

### Q3. 「事実更新の提案」が出てこない

**A**: 同じ entity (人物・場所・モノ) に対する矛盾事実を AI が検出した時のみ表示されます:
- 例: 「◯◯店オープン」(過去) → 「◯◯店閉店」(新)
- 単に新しい情報を補足する記事の場合は表示されません
- AI が確信を持てない場合は false (silent skip)

### Q4. AI チャットで「分かりません」と返ってくる

**A**: 知積の AI チャットは **保存した記事の内容のみに基づいて回答します**:
- 一般知識からの推測は行わない設計です (ハルシネーション抑制)
- 質問に該当する記事を保存していない場合、「分かりません」と回答します
- 関連記事を保存してから再度質問してください

### Q5. PDF の内容が抽出されない

**A**: 以下を確認してください:
- ファイルサイズが **5 MB 以下** か
- スキャン PDF (画像のみ) ではないか (現状 OCR 未対応)
- パスワード保護されていないか

### Q6. Safari Web Extension の自動保存が動かない

**A**: 以下を確認してください:
- iOS 設定 → Safari → 拡張機能 → 知積 が有効
- Web サイトのアクセス権限を「**すべての Web サイト**」に設定
- 設定 → Safari ツールバーで保存 → 「自動保存」が ON
- 同じタブで別 URL に遷移した場合は履歴 API hook で対応 (10 秒待つ)

### Q7. プライバシーは大丈夫?

**A**: 知積は **完全ローカルファースト** です:
- 記事の内容、AI への質問、すべてのデータは端末内に保管
- Apple Intelligence (on-device) で AI 処理、外部送信ゼロ
- アクセス権限を要求するのは「Safari Extension の URL アクセス」のみ
- 広告 SDK / アナリティクス / クラウド同期 は一切なし

### Q8. データのバックアップ / エクスポート

**A**: 現状、SwiftData の標準同期に従います (iCloud Drive 経由のアプリデータバックアップ)。専用のエクスポート機能は将来 spec で実装予定です。

### Q9. 削除した記事は完全に消えますか?

**A**: ライブラリで左 swipe → 「削除」で完全削除されます (ゴミ箱なし、即時削除)。関連 AI データ (要約、KeyFact、entity、Tag relationship) も SwiftData の cascade で同時削除されます。一度削除した記事は復元できません。

### Q10. 端末を変更したい (機種変更)

**A**: SwiftData は iCloud バックアップと連動しており、端末バックアップから復元すれば知積のデータも引き継げます。専用の同期機能は未実装です。

## ⚠️ 既知の問題

### Apple Intelligence 不可端末での体験

- AI 機能 (要約 / トピック発見 / 矛盾検出 / Chat) は Fallback 動作
- 体験は限定的、Apple Intelligence 対応端末を推奨

### Safari Web Extension の制約

- 同タブ内 SPA navigation は履歴 API hook で対応済 (一部サイトでは見逃しの可能性)
- iframe 内記事は保存対象外 (main frame のみ)

### PDF サポート

- スキャン PDF (画像のみ) は本文抽出されません
- パスワード付き PDF は非対応

## 🛠️ トラブルシューティング

### アプリが起動直後にクラッシュする

1. 端末を再起動
2. アプリを削除して再インストール (データは失われます)
3. それでも解決しない場合 → GitHub Issues で報告

### 記事を保存したのに表示されない

- ライブラリタブで pull-to-refresh
- ネットワーク状況確認 (記事の本文取得時にオンライン必要)
- 数分待っても表示されない場合 → GitHub Issues

### AI 処理が遅い

- 記事数が多い場合、初回 backfill で時間がかかります
- バッテリー / 充電状態によって BGTaskScheduler が遅延
- 5 分以上待っても結果が出ない場合 → GitHub Issues
