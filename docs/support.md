# Knowledge Base — サポート / Support

Knowledge Base は、読んだ記事を端末内 AI が自動で整理し、必要なときに思い出せる「あなた専用の第二の脳」です。

## お問い合わせ / Contact

ご質問・不具合報告・ご要望は **GitHub Issues** よりお願いします:

👉 **https://github.com/changch223/KnowledgeBase/issues**

For questions, bug reports, or feature requests, please open a GitHub Issue at the link above.

不具合報告では以下があると対応がスムーズです:
- 端末モデル (例: iPhone 17)
- iOS バージョン
- アプリのバージョン
- 再現手順 / 期待 vs 実際の動作

## よくある質問 / FAQ

**Q. 保存したデータはどこに保存されますか?**
すべてお使いの端末内に保存されます。iCloud 同期を ON にすると、あなた自身の iCloud にのみ同期され、開発者はアクセスできません。詳しくは [プライバシーポリシー](./privacy-policy.md) をご覧ください。

**Q. インターネット接続は必要ですか?**
記事を保存する際、本文・メタデータ取得のための通信を行います。要約・分類・チャットは端末内 AI (Apple Foundation Models) で処理されます。

**Q. 記事の保存方法は?**
Safari / Chrome / X などの共有メニューから「Knowledge Base」を選ぶだけで保存できます。

**Q. データを削除したい**
アプリ内から個別/一括で削除できます。アプリを削除すると端末内データは消去されます。iCloud 同期データは iOS の iCloud 設定から削除できます。
