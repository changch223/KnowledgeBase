# Knowledge Base

## 保存するだけ。AIが整理する。あなた専用の第二の脳。

---

## こんな悩みありませんか？

- 「**あの記事どこだっけ？**」と探す時間が積み上がっている
- 読んだ内容が **バラバラで全体像が見えない**
- タブを **開いたまま読まずに溜め込んでしまう**
- 自分で構造を作る Notion 系ツールが続かない
- 「保存して終わり」の Pocket 系ツールに罪悪感

---

## Knowledge Base が解決します

**読んだ記事を、AIが裏で1つの百科事典に編さんし続け、開くだけで自分だけの知識が育っていくのが見える、優しい第二の脳。**

あなたがやることは、**気になったものを保存するだけ**。あとは Apple Intelligence（端末内AI）が読んで、要点を抜き、テーマごとにまとめ、新しい記事が来たら既存の理解を更新します。

---

## あらゆるものを取り込める

| 取り込み方法 | 対応コンテンツ |
|---|---|
| Safari 共有シート | Webページ、PDF添付（Gmail 等） |
| Safari Web 拡張機能 | 閲覧中のページをワンタップ保存 |
| ＋ボタン — URL | URLを直接入力 |
| ＋ボタン — メモ | テキストを貼り付け |
| ＋ボタン — ファイル | PDF / txt / Markdown |
| ＋ボタン — 写真 | OCR で文字を読み取り |
| ＋ボタン — 音声 | m4a / mp3 等を文字起こし |

英語・中国語など外国語のコンテンツは、**日本語に翻訳してから**知識化します。

---

## AIが自動で知識に変える

- 要約（essence）と「重要な事実」を抽出
- タグと分野（カテゴリ）を自動で付与
- 複数の記事を横断した「**概念ページ（Wiki）**」を生成・更新
- 音声の固有名詞は用語集を使って自動補正
- 分類は使うほど自己学習（ユーザー修正を few-shot として記録）

---

## 3 つのタブで知識を活用

### iKnow タブ（起動時のデフォルト）
- 概念ごとの「超まとめ」を**答え先出し**で一覧
- 新着記事の横スクロール棚
- フィードに分野・タグのハイライト

### ライブラリ タブ
- 全記事を日付別に一覧
- 関連度ランキング付き検索

### AIチャット タブ
- 保存した記事だけを根拠に回答
- **番号付きの出典** `[1][2]` → タップで元記事へ
- 会話履歴を考慮したマルチターン対話

---

## 自分好みに調整できる

- **本文訂正**: 記事詳細の「訂正」ボタンから自然言語で指示（例: 「Claude Code を gloadcod と誤認識しています」）→ AIが該当箇所だけ修正 → 概念・タグも再生成
- **生成カスタマイズ**: まとめ方の方向性を指定（「技術的な詳細を重視」「短めに」など）
- **分類の確認**: 要確認タグに正しい分野を選ぶと、その修正が次回の分類に活かされる
- **概念ページ編集**: 本文を自由に書き換え・非表示・ピン留め

## プライバシー最優先・完全オンデバイスAI

| 項目 | 状態 |
|---|---|
| AI処理（要約・抽出・チャット・翻訳） | ✅ 端末内のみ（Apple Intelligence） |
| 音声文字起こし | ✅ 端末内のみ（Speech フレームワーク） |
| 開発者へのデータ送信 | ❌ ゼロ |
| アカウント登録・ログイン | ❌ 不要 |
| 広告 SDK / アナリティクス | ❌ なし |
| iCloud 同期 | ✅ 任意（ユーザーのプライベート DB のみ） |

---

## 必要な環境

- **iOS 26.4 以降** / iPadOS 26.4 以降
- **Apple Intelligence 対応端末**（非対応端末でも保存・閲覧・検索は利用可）
- 主言語: **日本語**（UI・AI生成・翻訳のすべてが日本語ファースト）

---

## 競合との違い

| サービス | Knowledge Base との違い |
|---|---|
| Pocket / Instapaper | 保存だけじゃない、AIが自動で知識に変える |
| Notion | 自分で構造を作らない、AIに任せる |
| Readwise | ハイライト単発ではなく、テーマ横断で概念ページが育つ |
| ChatGPT / Claude | 自分が蓄積した知識（ソース付き）に対話できる |
| Obsidian | vault・グラフの専門知識不要、iCloud 同期付きローカルファースト |
