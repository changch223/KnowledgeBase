# iKnow プライバシーポリシー

**最終更新日**: 2026-05-24
**運営者**: Chia-Wei Chang
**Contact**: https://github.com/changch223/KnowledgeTree/issues

## 1. このアプリは何を収集しないか

iKnow (以下「本アプリ」) は、**ユーザーのデータを一切外部サーバーに送信しません**。

### 収集しない情報
- 個人を識別できる情報 (氏名、メールアドレス、電話番号、Apple ID 等)
- 利用解析 / クラッシュレポート / トラッキング情報
- 広告識別子 (IDFA)
- 位置情報
- 連絡先 / カレンダー / 写真 / マイク

## 2. 端末内に保存される情報

以下の情報は **すべてあなたの端末内のみ** に保存され、外部に送信されません:

- あなたが保存した記事の URL / タイトル / 本文
- 記事から AI が抽出した知識 (要約、KeyFact、entity)
- AI チャットの履歴 (質問と答え)
- 概念ページ (人物・モノ・概念の自動統合知識)
- 行動履歴 (「✓ わかった」「🤔 もっと」等の学習操作記録)
- アプリ設定

これらは iOS の SwiftData により端末内のサンドボックスに暗号化保存されます。

## 3. AI 処理について

本アプリは **Apple Intelligence (Foundation Models)** を利用して要約・分類・チャット機能を提供します:

- AI 処理は **すべて端末内 (on-device)** で完結します
- Apple のサーバーや外部の AI サービス (OpenAI / Anthropic 等) には一切送信されません
- Apple Intelligence の動作については [Apple のプライバシーポリシー](https://www.apple.com/legal/privacy/) をご確認ください
- 非対応端末では AI 機能は利用できませんが、記事保存・閲覧は引き続き可能です

## 4. 外部リンク

本アプリの「不具合の報告 / 要望」リンクは GitHub (https://github.com/changch223/KnowledgeTree) に遷移します。GitHub の利用は [GitHub のプライバシーポリシー](https://docs.github.com/en/site-policy/privacy-policies) が適用されます。

「プライバシーポリシー」(本ページ) は GitHub 上に公開されています。アクセスは GitHub のサーバーログに記録されますが、本アプリ側は何も追跡しません。

## 5. データの削除

- アプリ内の記事 / 答え / 概念ページは個別に削除可能です (長押し / swipe / 詳細画面 toolbar)
- アプリをアンインストールすると、端末内の全データが完全に削除されます
- iCloud / 外部バックアップは現バージョンでは利用していません

## 6. 子供のプライバシー

13 歳未満の利用者から意図的にデータを収集することはありません (そもそも何も収集しないため)。

## 7. 変更通知

本ポリシーが変更された場合、本ページの「最終更新日」が更新されます。重要な変更がある場合はアプリ内の「変更履歴」セクションで通知します。
